## TL;DR

"Now with limitless backgrounds. A better way to brag your stats for free. Stop sharing boring screenshots. Instantly turn your achievements into beautiful, highly-customizable images that command attention."

## Tech Stack

- **Framework**: Next.js 16.1.6 (App Router) / React 19
- **Styling**: Tailwind CSS v4, utility classes configured with tailwind-merge and clsx
- **UI Components**: shadcn/ui, Radix UI primitives, Lucide React icons
- **Animations**: motion (Framer Motion) and tw-animate-css
- **Utilities**: html-to-image (for exporting designs), next-themes (for dark/light mode)
- **Fonts**: Custom configured via next/font/google including Faculty Glyphic, Fira Code, and several other artistic fonts.
- **SEO Tooling**: next-sitemap

## Features

Bragly provides a robust and clean interface with the following capabilities:

- **Free Forever**: No credit card required. Generate unlimited visuals without watermarks.
- **Easy Customization**: Tweak every pixel. Adjust padding, inset shadows, border radii, and curate fonts with instant visual feedback.
- **Modern Design**: Pixel-perfect aesthetic designed to stand out, encapsulated in a premium user experience (UX).
- **Unlimited Backgrounds**: An exhaustive collection of high-quality mesh gradients, solid colors, and abstract image/patterns.
- **Modern Typography**: Specialized geometric and serif fonts configured for sharp rendering and readability.
- **Optimized Export**: Download creations in pristine high-resolution PNG format, perfect for Open Graph images, Twitter/X posts, and structured data imagery.
- **Instant Processing**: Entirely client-side rendering engine (direct in-browser) for rapid feedback, improved privacy, and zero server-delay.
